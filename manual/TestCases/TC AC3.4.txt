TC AC3.4

Description:
User can clear "Created On" filter using the "Clear All" button

Precondition:
TopicA created today
TopicB created week

Steps:
1. Login to the application
2. Navigate to "Topic Hub"
3. Click on the "Created on" date picker button
	-> Date Picker appears
4. Select first Date today
5. Select second Date yesterday
	-> TopicA appears in the list
6. Date range is selected (displayed below selection button)
7. In the applied filter section click on the "Clear All" button for the applied filter

Expected: 
TopicA and TopicB appear in the list
Filter is cleared
Reopening the filter again shows no dates selected
